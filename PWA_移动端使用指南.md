# PWA 移动端使用指南

## 快速开始

### 1. 启动应用

**Windows:**
```bash
# 使用启动脚本（推荐）
run_pwa.bat

# 或手动启动
streamlit run app.py --server.headless=true --server.address=0.0.0.0 --server.port=8501
```

**Mac/Linux:**
```bash
streamlit run app.py --server.headless=true --server.address=0.0.0.0 --server.port=8501
```

### 2. 获取电脑 IP 地址

**Windows:**
```cmd
ipconfig
```
找到 "IPv4 地址"，如 `192.168.1.100`

**Mac/Linux:**
```bash
ifconfig
# 或
ip addr show
```

### 3. 手机访问

1. 确保手机和电脑在**同一 Wi-Fi 网络**
2. 在手机浏览器输入：`http://电脑 IP:8501`
   - 例如：`http://192.168.1.100:8501`

## 添加到主屏幕

### iPhone / iPad (Safari)

1. 用 Safari 打开应用
2. 点击底部 **分享** 按钮（方形带箭头）
3. 滑动找到并点击 **"添加到主屏幕"**
4. 点击右上角 **"添加"**
5. 主屏幕上会出现应用图标，点击即可打开

### Android (Chrome)

1. 用 Chrome 打开应用
2. 点击右上角 **菜单** (三个点)
3. 点击 **"安装应用"** 或 **"添加到主屏幕"**
4. 确认安装
5. 主屏幕上会出现应用图标

### Android (Samsung Internet)

1. 打开应用
2. 点击底部 **菜单** (三条线)
3. 点击 **"添加到页面"**
4. 选择 **"添加"**

## 移动端优化详情

### 布局优化
- **字体大小**: 移动端自动调整标题和正文字体
- **按钮**: 全宽显示，更易点击
- **侧边栏**: 280px 宽度，适合手机
- **图表**: 自适应宽度，支持触摸缩放
- **表格**: 横向滚动，不溢出屏幕

### 性能优化
- **缓存**: Service Worker 缓存静态资源
- **视口**: 禁止用户缩放，优化触摸体验
- **主题色**: 4F46E5 紫色主题，匹配品牌

## 常见问题

### Q: 手机访问不了？
**A:** 检查以下几点：
1. 手机和电脑是否在同一 Wi-Fi 网络
2. 电脑防火墙是否允许 8501 端口
3. IP 地址是否正确

### Q: 添加到主屏幕后无法更新？
**A:**
1. 删除主屏幕上的应用
2. 重新访问网页并添加
3. 或者等待自动更新（最多 24 小时）

### Q: 离线能用吗？
**A:**
- 当前版本支持基础离线缓存
- 但成绩数据需要联网获取
- 完整离线模式开发中
