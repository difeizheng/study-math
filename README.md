# 学生成绩分析系统

一个深度分析小孩数学成绩的信息系统，基于**人教版小学数学知识点体系**，覆盖 1-3 年级上下学期。

## 功能特点

### 📈 成绩趋势分析
- 可视化展示各学期成绩变化
- 分数分布饼图和柱状图
- 平均分、标准差等统计指标

### 🧠 知识点深度分析
基于人教版小学数学知识点体系，深度分析：
- **数与代数** 🔢 - 数的认识、计算能力、运算律
- **图形与几何** 📐 - 空间观念、几何直观、图形认识
- **统计与概率** 📊 - 数据分析观念、统计图表
- **综合与实践** 🔧 - 应用意识、创新意识、问题解决

### 📋 诊断报告
- 自动生成个性化学习诊断报告
- 薄弱知识点识别
- 针对性学习建议
- 支持报告下载

### 👥 多学生对比
- 支持 2-5 名学生同时对比
- 知识领域雷达图
- 成绩统计对比

### ⚠️ 成绩预警
- 退步预警：监测成绩连续下降
- 波动预警：检测成绩稳定性
- 低分预警：及时发现学习困难

### 📊 班级分析
- 班级整体统计
- 分数段分布
- 成绩分布直方图
- 详细数据表

### 🔬 宏观分析
- SAI 学业发展综合指数
- 定量分析（集中趋势、离散程度、分布形态）
- 定性评价（学业水平、稳定性、发展趋势）
- 两人对比模式

### 📕 错题追踪本（v3.0 新增）
- 记录学生每次考试的错题
- 艾宾浩斯遗忘曲线复习提醒
- 举一反三练习推荐
- 错题本导出

### 🕸️ 知识点关联图谱（v3.0 新增）
- 展示知识点的前后置依赖关系
- 分析薄弱知识点的影响范围
- 学习路径推荐
- 根本原因分析

### 🌟 能力成长档案（v3.0 新增）
- 五大核心素养评估
  - 数感 🔢
  - 符号意识 🔣
  - 空间观念 📐
  - 数据分析观念 📊
  - 推理能力 🧠
- 能力雷达图
- 成长报告导出

### 📝 学习习惯分析（v3.0 新增）
- 错误类型分析（粗心 vs 知识性）
- 六大习惯维度评分
- 个性化改进建议
- 习惯变化趋势追踪

### 🏫 班级学情看板（v3.0 新增）
- 教师视角的班级整体分析
- 知识点掌握率热力图
- 需要关注的学生识别
- 班级学情报告导出

### 📄 智能组卷（v3.0 新增）
- 根据薄弱知识点自动生成练习卷
- 三种试卷类型（专项突破/基础练习/单元检测）
- 支持 Word/Markdown 格式导出
- 附带参考答案和答题卡

## 知识点体系

### 一年级上册
- 准备课、位置
- 1-5 的认识和加减法
- 认识图形 (一)
- 6-10 的认识和加减法
- 11-20 各数的认识
- 认识钟表
- 20 以内的进位加法

### 一年级下册
- 认识图形 (二)
- 20 以内的退位减法
- 100 以内数的认识
- 认识人民币
- 100 以内的加法和减法 (一)

### 二年级上册
- 长度单位
- 100 以内的加法和减法 (二)
- 角的初步认识
- 表内乘法 (一)、(二)
- 观察物体 (一)
- 认识时间

### 二年级下册
- 数据收集整理
- 表内除法 (一)、(二)
- 图形的运动 (一)
- 混合运算
- 有余数的除法
- 万以内数的认识

### 三年级上册
- 时、分、秒
- 万以内的加法和减法
- 测量
- 倍的认识
- 多位数乘一位数
- 长方形和正方形
- 分数的初步认识

### 三年级下册
- 位置与方向
- 除数是一位数的除法
- 复式统计表
- 两位数乘两位数
- 面积
- 年、月、日
- 小数的初步认识

## 使用说明

1. **启动系统**
   ```bash
   streamlit run app.py
   ```
   系统会在浏览器中自动打开，默认端口为 8501。

2. **选择学生** - 在左侧边栏选择要分析的学生

3. **选择分析模式** - 系统提供 13 种分析模式：
   - 📈 成绩趋势分析：查看成绩走势和分布
   - 🧠 知识点深度分析：查看各知识点掌握情况
   - 📋 诊断报告：生成并下载详细报告
   - 👥 多学生对比：对比多个学生的表现
   - ⚠️ 成绩预警：智能监测学生学习状态
   - 📊 班级分析：查看班级整体学习情况
   - 🔬 宏观分析：SAI 学业发展综合指数
   - 📕 错题追踪本：记录和管理错题
   - 🕸️ 知识点关联图谱：分析知识依赖关系
   - 🌟 能力成长档案：五大核心素养评估
   - 📝 学习习惯分析：识别粗心 vs 知识性错误
   - 🏫 班级学情看板：教师视角的班级分析
   - 📄 智能组卷：自动生成针对性练习卷

4. **查看分析结果** - 系统自动识别薄弱环节并给出建议

5. **下载报告** - 可下载诊断报告、错题本、成长档案等用于后续学习规划
